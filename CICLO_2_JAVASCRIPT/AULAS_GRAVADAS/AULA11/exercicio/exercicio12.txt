1. O que o manipulador ou função replace() efetua em uma string? 
    RESP.: É um metodo que troca ou string dentro da string

2. O que o manipulador ou função sort() efetua em um array?
    RESP.: Essa função ordena os elementos de um array em ordem crescente, 
        nos elementos numericos essa função não é precisa.

3. É correto dizer que o JS é uma linguagem orientada a objetos? 
    RESP.: Não, apesar com as novas atualizações tem uma implementação orientada objetos,
    que cria e manipula objetos.

4. Como podemos definir o operador new?
    RESP.: O operador new é uma palavra reservado que criar um novo objeto.
